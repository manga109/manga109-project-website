Annotations

Manga109
Annotations.

Manga109 の全ての作品には，キャラクター・顔・全身・コマ枠・テキストの詳細なアノテーションが付与されています．アノテーションの利用を希望する場合には，Manga109 本体と同じく Download ページよりご連絡ください．

01 / Structure

アノテーションの内容．

漫画1巻と1つの XML ファイルが対応しています．文字コードは UTF-8 で，各ファイルの大まかな構造は次のようになっています．

  • book作品タイトル
    • charactersその漫画に登場するキャラクターのリスト
      • character
      • character
      • …
    • pagesページ（見開き）のリスト
      • page
      • page
      • …

character には各キャラクターの id と名前の対応が記述されています．page には各ページの情報（見開きページ番号，画像サイズ）とそのページ内のオブジェクトが記述されています．オブジェクトは4種類あり，全てに id と矩形領域（xmin, xmax, ymin, ymax）が付与されています．キャラクターやオブジェクトの id は8桁の16進数となっており，本データセット内で一意です．

01

face

キャラクターの顔．

キャラクター id が付与されています．
02

body

キャラクターの全身．

キャラクター id が付与されています．
03

text

活字のテキスト，一部の描き文字．

テキストの内容が記述されています．
04

frame

コマ枠．

固有情報なし．

詳細なアノテーションの基準は アノテーション基準 ↗ をご覧ください．
